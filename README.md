# SceneSwitchWindow
![UI Preview](https://i.imgur.com/TcWMdH9.png)
A GUI Window that shows the list of scenes added in the build settings.

## Quick Package Install

#### Using UnityPackageManager (for Unity 2019.3 or later)

Click '+' -> Choose 'Add package from git URL' -> enter https://github.com/ta-david-yu/Unity-SceneSwitchWindow.git#upm
